## Events  

### `SplitterDeployed`

```javascript
    event Splitter_Deployed(bytes32 indexed ArtistId, address indexed RoyaltyAddr, address indexed Factory, address[] payees, uint256[] shares)
```

Event is emitted by the deploySplitter method.

| Parameter | Type | Description |
|:---|:---:|---:|
| ArtistId `indexed` | bytes32 | Indexed - The artist identifier. |
| RoyaltyAddr `indexed` | address | Indexed - The RoyaltySplitter address.  |
| Factory `indexed` | address | Indexed - The Factory address. |
| payees  | address[] | An array of accounts. |
| shares  | uint256[] | An array of shares. |

<br />  

## methods  

### `deploySplitter`

```javascript
    function deploySplitter(bytes32 _artistId, uint256 _salt, address[] _payees, uint256[] _shares) external nonpayable returns (address)
```

Restricted method used to deploy a new PaymentSplitter contract dedicated to an artist/band.

*Restricted with OPERATOR_ROLE.Will emit the event { Splitter_Deployed }.*

| Parameters | Type | Description |
|:---|:---:|---:|
| _artistId | bytes32 | The artist Identifier. |
| _salt | uint256 | The salt used in the deployment. |
| _payees | address[] | An array of accounts. |
| _shares | uint256[] | An array of shares. |


| Returns| Type | Description |
|:---|:---:|---:|
| _0 | address | undefined |
